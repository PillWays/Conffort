Documentación Tienda Online

  • Manual de uso para el programa de aplicación web tienda online de venta de tenis Conffort.
  • La aplicación web ofrece un servicio de venta para tenis, el usuario se registra en el sitio web, inicia sesión dentro del sitio y muestra el catalogo disponible de tenis.

Desarrollado con:

  • Python 3.8.3
  • Django 3.0.7
  • SQLite 3.32.2
  • HTML5
  • CSS3
  • Fullpaje.js
  • Bootstrap4
